บันไดมาตรวัด — ใช้ 3 คำถามนี้ไล่ทีละขั้น
  1. จัดอันดับ (มากกว่า/น้อยกว่า) ได้ไหม? ไม่ได้ → Nominal
  2. ได้ แต่ "ช่องว่าง" ระหว่างขั้นเท่ากันไหม? ไม่เท่า/ไม่รู้ → Ordinal
  3. เท่ากัน แล้วมี "0 จริง" (true zero = ไม่มีของสิ่งนั้น) ไหม? ไม่มี → Interval · มี → Ratio

เปิดดูตารางสรุป: มาตรวัด → ค่ากลางที่ใช้ได้
มาตรวัดลักษณะค่ากลางที่ใช้ได้ตัวอย่างเภสัช/คลินิก
Nominalป้ายชื่อ จัดอันดับไม่ได้Mode เท่านั้น (+ นับ/สัดส่วน)หมู่เลือด, เพศ, ชนิดยา
Ordinalเรียงอันดับได้ แต่ช่องว่างไม่เท่าMedian, Mode (ไม่ใช้ mean)ADR grade, CKD stage, Likert
Intervalช่วงเท่ากัน แต่ไม่มี true zeroMean, Median, Mode (อัตราส่วนไม่มีความหมาย)อุณหภูมิ °C, ปี ค.ศ.
Ratioช่วงเท่ากัน + มี true zeroทั้งหมด (+ %CV, geometric mean, อัตราส่วน)ความเข้มข้นยา, น้ำหนัก, จำนวนเม็ด
หมายเหตุ: เรื่อง "ค่ากลางใช้อะไรได้" Interval กับ Ratio เหมือนกัน — ต่างกันที่ การคำนวณอัตราส่วน/%CV/GM ซึ่งต้องมี true zero (ต่อบล็อก E–F)
