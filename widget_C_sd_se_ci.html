SD เทียบ SE เทียบ 95% CI — เห็นภาพ

Widget C · Week 1 · เลื่อน n แล้วสังเกต: SD (การกระจายของข้อมูล) แทบนิ่ง · SE และ CI หดเข้าเมื่อ n มากขึ้น · ข้อมูลจำลองจาก Sample 1 (µ = 48.7, σ = 5.4 mg)

SD = ข้อมูลกระจายแค่ไหน (อธิบายเม็ดยารายตัว)  ·  SE = ค่าเฉลี่ยแม่นแค่ไหน = SD/√n  ·  95% CI = ช่วงที่น่าจะครอบ µ จริง = mean ± t·SE
กดดูสิ่งที่ควรสังเกต
1) ลาก n ให้น้อย (เช่น 4) แล้วกด "สุ่มตัวอย่างใหม่" ซ้ำ ๆ — เส้นค่าเฉลี่ย (เขียวทึบ) จะเด้งไปมารอบ µ จริง (เส้นประ) มาก · รอยจาง ๆ ด้านบนคือค่าเฉลี่ยจากการสุ่มที่ผ่านมา กระจายกว้างราว ±2·SE
2) ลาก n ให้มาก — ค่าเฉลี่ยแทบไม่เด้ง, แถบ SE/CI หดแคบ แต่แถบ SD ยาวเท่าเดิม
3) กด "วัดทั้ง lot" — SE = 0, CI ยุบเป็นจุด (รู้ µ เป๊ะ) แต่ SD ยังอยู่ เพราะเม็ดยายังหนักไม่เท่ากัน
