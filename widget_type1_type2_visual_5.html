Type I vs Type II — เห็นภาพ

Week 3 · ลากเส้นตัดสินใจ แล้วดูว่า error สองแบบแลกกันอย่างไร
เส้นซ้ายคือโลกที่ ยาไม่มีผลจริง · เส้นขวาคือโลกที่ ยามีผลจริง
Type I = เตือนภัยลวง → ตะโกนว่า "ยาได้ผล!" ทั้งที่จริง ๆ มันแค่บังเอิญ
Type II = พลาดของจริง → สรุปว่า "ไม่ได้ผล" ทั้งที่จริง ๆ ยามีผล
วิธีอ่านกราฟนี้ · สมมติฐานที่ใช้
  • การทดสอบ: ทางเดียว (one-sided) ใช้การประมาณแบบ z (สมมติว่า σ ทราบค่า = 10). t-test ใช้ตรรกะเดียวกัน เพียงแต่เส้นโค้งเป็น t และค่าวิกฤตสูงกว่าเล็กน้อยเมื่อ n เล็ก
  • H₀: μ(ยาใหม่) − μ(เดิม) = 0 — ยาไม่มีผล → เส้นโค้งซ้าย
  • H₁: μ(ยาใหม่) − μ(เดิม) > 0 — ยาลดความดันได้มากกว่า → ทิศปฏิเสธอยู่ทางขวา
  • แกนนอน: ผลต่างเฉลี่ยที่สังเกตได้ (mmHg) · ความกว้างเส้นโค้ง = SE = σ/√n
  • α = พื้นที่หาง H₀ ทางขวาของเส้น · β = พื้นที่ Ha ทางซ้ายของเส้น · power = 1−β
↔ ลากเส้นประแนวตั้ง (เส้นตัดสินใจ) ได้โดยตรง
